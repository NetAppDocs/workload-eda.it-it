= Scopri di più su NetApp Workload Factory per EDA
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Workload Factory per EDA ti aiuta a ottimizzare FSx per ONTAP su più file system. È possibile ottimizzare le prestazioni e ridurre i costi operativi automatizzando i parametri di archiviazione, analizzando i vincoli prestazionali e ottenendo informazioni dettagliate sui progetti EDA.

È progettato per integrarsi con i framework Infrastructure as Code (IaC).

Workload Factory per EDA fornisce dashboard, framework e utilità di automazione dell'archiviazione che aiutano a gestire più file system FSx for ONTAP .

Offre quanto segue:

* link:task-use-overview-dashboard.html["dashboard overview"]: fornisce una vista centralizzata del cluster per aiutarti a monitorare lo stato, la capacità e il throughput del cluster FSx for ONTAP, determinare dove posizionare i nuovi volumi e prendere decisioni informate sulla scalabilità della capacità e del throughput.
* link:task-monitor-latency.html["Analisi della latenza FSx"]: monitora proattivamente le performance di latenza in lettura e scrittura dei volumi con soglie di avviso e di evento critico configurabili. Include un'analisi di base automatizzata utilizzando le metriche ONTAP QoS delay center.
* link:task-configure-dashboard.html["Dashboard del progetto"]: fornisce una visione centralizzata del consumo di storage nei tuoi file system FSx for ONTAP per aiutarti a pianificare, valutare i costi e raccogliere informazioni sull'utilizzo dello storage nei tuoi progetti.
* link:learn-eda-workloads.html["CI/CD"]: semplifica e riduce i tempi di compilazione del software sfruttando le capacità di clonazione del volume di FSx for ONTAP .
